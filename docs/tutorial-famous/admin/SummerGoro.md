# SummerGoro(五郎)

## 介绍

2017年开始进行开服活动，IMC的服务器硬件支持及运维，，是划水多年的咸鱼鸽子一枚。

五郎所创建的SMGoro-Server为IMC服务器前身，后他也为开发者提供机器，gg66用此在2022年5月1日开启了wss服务器，标志着IMC服务器的正式创立

### 网页先驱

五郎在[MineBBS](https://www.minebbs.com/resources/eaglecraft-minecraft.4151/)为广大腐竹提供了免费的EaglerCraft汉化，且详细介绍如何使用EaglerCraft
