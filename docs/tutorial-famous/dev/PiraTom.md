# PiraTom

## 简介

一个只会做小游戏的疯子，负责IMC几乎所有玩法的开发和后续维护

音游爱好者，前身为BCM腐竹,开发过盾牌音游、Cube Rhythm等项目

### 鸭子

`PiraTom`原先叫`Eroslon_Dusk`，有玩家误看做`Eroslon_Duck`，后大家便叫**鸭子**了

### 五郎之家

BCM服务器主城左侧的一个房间，摆放了三只五郎NPC和各种五郎梗图

门前有一行悬浮字：**打倒五郎暴政，世界属于直连**
