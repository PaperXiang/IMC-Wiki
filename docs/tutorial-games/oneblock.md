# 单方块(oneblock)

:::tip 作者
本页面由Luke_0115及OF_666awa编写
:::

    单方块生存，顾名思义，玩家可以仅靠脚下的一个会生成不同方块的方块来发展，是风靡全球的玩法之一.  

    而在IMC之中，在原本的基础上增加了许多新的机制，于`2025.04.04`开启新周目，已有数百位玩家游玩.

## 基础玩法

### 概要

    玩家在创建世界后，会获得一些石质工具，并且出生在一个草方块上，依靠这个方块来发展.

    每个玩家都有一个独立的世界，并非共用同一个世界，互不互通.

## 新机制

### 维度

![进入传送门后的gui](https://image.8aka.org/file/1746276167082_download.png)

    与传统玩法不同的是，主世界的方块出现的方块类型永远不变，而不是挖够足够数量的单方块变换下一个群系.取而代之的是传送门，可以凭此进入更多维度，以及挖掘相应维度特有的单方块，获得更多的物资.

    需要注意的是，主世界的大小为512x512，其他维度为128x128

### 韵律转化桌

![韵律转换桌的gui](https://image.8aka.org/file/1746276373658_download.png)

**合成方法**：放置一个讲台，在上面放一个唱片机.

**使用方法**：用任何唱片右键唱片机，可随机获得0.1-1的唱片能量.

**用途**：使用除唱片以外的物品可打开韵律转化桌的gui界面，可展示这台韵律转化桌的唱片能量，可兑换物品.(50兑换一个传送门器)
        
          ——该物品在铁砧修改为坐标后，丢进传送门后即可.进入该传送门可以传送到对应的位置.
