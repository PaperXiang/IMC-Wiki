# 枪战大厅

枪战大厅是枪战游戏的中心基地。玩家可以在此处购买装备、训练射击、查看排行榜等。


## 商店

玩家在商店可以购买强力武器。建议购买前先去靶场试试手感，并参考其他玩家建议。

总共有8类物品可供选择：步枪、狙击枪、冲锋枪、机枪、霰弹枪、手枪、近战武器、投掷/特殊武器。

喊话用的喇叭也可以在这里购买。

![商店大门](<https://image.8aka.org/file/1746369411994_屏幕截图 2025-05-04 223630.png>)

![商店](https://image.8aka.org/file/1746288012803_download.png)


## 预设

玩家可在预设中搭配自己惯用的装备，视情况自主选择。

右击快捷栏中的箱子即可设置预设。

每个玩家最多保存4个预设，**至少要有1个正确的预设才能开始游戏**。

![预设](https://image.8aka.org/file/1746287950634_download.png)


## 游戏统计

游戏统计功能可以展示玩家游戏次数、最高分、胜场次等。

右击快捷栏中的书即可打开游戏统计。

![统计](https://image.8aka.org/file/1746288054524_download.png)


## 开始游戏

右击快捷栏中的指南针，打开后选择一个小游戏频道进入。

![选择](https://image.8aka.org/file/1746288021359_download.png)


## 排行榜

排行榜显示玩家在全服的游戏情况。

![排行榜区域](<https://image.8aka.org/file/1746370007187_屏幕截图 2025-05-04 224626.png>)

![排行榜](https://image.8aka.org/file/1746288134068_download.png)

:::tip 硬币
排行榜上的硬币数量参考价值不大，因为硬币的消耗会导致实际情况并不真实。
:::


## 靶场

靶场可以帮助玩家练习射击、选择合适的枪械。

玩家可在靶场旁的武器库选取自己想要练习的武器，对准靶子射击。

旁边的近战武器也可领取练习。

在靶场，弹药可以通过 `/resetammo` 来重置。

![靶场](<https://image.8aka.org/file/1746370332087_屏幕截图 2025-05-04 225152.png>)


## 近战武器

在大多数模式和生化模式的人类，近战武器的使用讲究技术。

手持近战武器按下右键，屏幕上会出现一个蓄力进度条。

在进度条靠近中间的时候按下左键攻击，可以打出更高的伤害。

**必须先按下右键，近战武器才能打出伤害。**

![近战](<https://image.8aka.org/file/1746370765780_屏幕截图 2025-05-04 225715.png>)


:::warning 连续点击
连续点击右键和左键，屏幕上会出现跟错过攻击时间一样的 **Miss** 字样，并且无法打出任何伤害。必须稍事等待才能打出有效伤害。
:::
