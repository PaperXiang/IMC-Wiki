## 阵营

![倒计时](https://image.8aka.org/file/1746286752943_download.png)

**人类**: 通过枪械，隐蔽点位等保护自己不被僵尸击杀，直到倒计时结束

**僵尸（母体）**：通过手中的斧子击杀人类（斧子有特殊效果，后文会提到），使得所有人类被僵尸感染则胜利

上方计数条会显示当前双方人数，胜利局数，本局剩余时长等

其中人类或僵尸双方率先达成胜利5(失落之城为3)次胜利时结束当局游戏并结算，下文会具体陈述。


## 开始

![人数不足](https://image.8aka.org/file/1746286773769_download.png)

一般来说，在生化模式下，人数到达**6**后会自动开启**1分钟**倒计时，当人数小于6时，停止计时但时间保留，计时结束后，一局开始。

曾经被管理更改成32和99
:::

当计时器归零并且有人类存活时，**人类方获胜**，每个存活的人类获得**100积分**

当管理终止游戏进程时，游戏直接结束，无视当前场上情况
:::

## 积分结算

当一局游戏结束后 系统会根据积分数量计算并得出硬币数量和获得经验数量，其中：

**积分区域**：

- 造成100点伤害获得1积分（向下取整）

- 远程击杀一只普通僵尸获得20积分，母体则为50积分

- 近战击杀僵尸/母体（不建议一般玩家这么做）获得150/300积分

- 僵尸感染人类获得50积分

**倍率区域**：基础倍率1.0，每次作为人类存活下来一局倍数+0.1

**最终的金币和经验值**: `基础积分 / 2 x 倍率`



## 效果 

### 迅捷

使用后获得**10s加速** 冷却**30s**

### 缴械

使用后向视角指向的方向发射一枚粘液球，当粘液球与人类判定箱接触时，该人类失去物品栏内物品5s（包括变身器），并获得min(score/20,30)分，向下取整

### 幻术

效果 使用后向视角指向的方向发射一枚雪球（速度略快于粘液球），击中人类后使得人类缓慢5s，冷却30s，并获得min(score/20,30)分，向下取整

### 坚固

使用后立刻获得抗性提升IV 10s，冷却30s

### 隐身

使用后获得20s隐身并暂时失去装备，被击中后失去该效果，结束后返还装备，冷却30s

### 变身器

当满足以下条件时，给予在场人类**变身器**

- 1.剩余时长小于1分30秒

- 2.存活人数小于5人

- 3. `90-10 x 人数＜0`

**效果**：使用后在近战和远程中二选一

- **近战**：获得一把刀，血量在下面最高血量基础上x1.5

- **远程**：获得榴弹炮，准度低的吓人，伤害高的吓人

:::tip 获取
当前版本无法在其他玩法获得本枪械
:::

最高血量变为**9000**

## 终极时刻

所有单位获得发光，存活玩家**回满血量**, 僵尸血量从400提高到**1000**，获得**力量**效果，伤害翻3倍，人类变化见变身器
